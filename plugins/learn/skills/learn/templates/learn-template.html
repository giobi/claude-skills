ls -la

Mostra tutti i file inclusi quelli nascosti, in formato lista dettagliata con permessi, owner e dimensioni.

Esempio:
$ ls -la /home/user/projects/
total 24
drwxr-xr-x  4 user user 4096 Mar 24 10:30 .
drwxr-xr-x 12 user user 4096 Mar 24 09:15 ..
-rw-r--r--  1 user user  220 Mar 24 10:30 .env
drwxr-xr-x  8 user user 4096 Mar 24 10:28 .git
-rw-r--r--  1 user user 1240 Mar 24 10:30 README.md

cd /path/to/directory

Cambia la directory corrente. Usa cd - per tornare alla precedente, cd ~ per la home.

Esempio:
$ pwd
/home/user
$ cd /var/log/nginx/
$ pwd
/var/log/nginx
